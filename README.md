# PokéTeams

PokéTeams is a front-end web application built using React and Redux. This site allows the user to create a Pokémon Team, as well as view more information about each Pokémon.

## Tech Stack
1. JavaScript
2. React.JS
3. React-Redux
4. HTML/CSS
5. Chakra-UI(?)
